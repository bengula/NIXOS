# NIXOS Configuration

This repository contains a backup of my NixOS configuration files, managed using Nix Flakes.

## Configuration Files

### `flake.nix`

This is the entry point for the NixOS configuration. It defines the inputs and outputs of the system, ensuring a reproducible build.

- **Inputs**:
  - `nixpkgs`: The primary package repository, pinned to the `nixos-unstable` channel.
  - `flake-utils`: A helper library for creating flakes.

- **Outputs**:
  - `nixosConfigurations.Erudite`: Defines the NixOS system configuration for the "Erudite" host.

### `configuration.nix`

This file contains the main system configuration. Below is a breakdown of the key settings:

- **Bootloader**:
  - GRUB is enabled as the bootloader on `/dev/sda`, with OS prober to detect other operating systems.

- **Networking**:
  - The hostname is set to "Erudite".
  - `NetworkManager` is enabled for managing network connections.
  - Google's public DNS servers (`8.8.8.8`, `8.8.4.4`) are configured.

- **Time & Locale**:
  - The timezone is set to `Africa/Nairobi`.
  - The default locale is `en_US.UTF-8`.

- **Display/GNOME**:
  - The X server and GDM (GNOME Display Manager) are enabled.
  - The GNOME desktop environment is enabled, with some default applications (Epiphany, GNOME Tour, GNOME Music) removed.

- **Sound**:
  - PipeWire is configured as the audio server, with support for ALSA and PulseAudio.

- **Users**:
  - Two users are configured:
    - `bengula`: A normal user with `thunderbird` installed.
    - `bebe`: A normal user with `tree`, `obsidian`, `chromium`, `homebank`, and `gnome-extension-manager` installed.
  - Auto-login is enabled for the `bengula` user.

- **System Packages**:
  - A list of system-wide packages is installed, including `neovim`, `git`, `libreoffice`, `vscodium`, and `python3`.

- **Garbage Collection**:
  - Automatic garbage collection is scheduled to run daily at 03:15, deleting generations older than 7 days.

### `hardware-configuration.nix`

This file is automatically generated by `nixos-generate-config` and contains hardware-specific settings. It is generally not recommended to modify this file directly.

- **Kernel Modules**:
  - Specifies necessary kernel modules for boot, such as `ahci` and `sd_mod`.
- **File Systems**:
  - Defines the root filesystem (`/`) and swap device.
- **Host Platform**:
  - Sets the host platform to `x86_64-linux`.
